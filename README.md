# sterne-sdl2
[Sterne](https://doctor-rd.github.io/sterne-sdl2/)

#### Cloning
`$ git clone --recurse-submodules https://github.com/doctor-rd/sterne-sdl2.git`\
`$ cd sterne-sdl2`

###### Build for your local machine
`$ make`\
`$ ./sterne`

###### Build for Web
`$ make -f Makefile.emscripten`\
`$ cd web`\
`$ python3 -m http.server`
